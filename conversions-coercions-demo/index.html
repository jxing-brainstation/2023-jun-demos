Review the 7 Primitives

  1. string
  2. number
  3. boolean
  4. null
  5. undefined
  6. bigint
  7. symbol

everything else is an object (objects, functions, arrays, etc.)

Explicit conversions (type casting): there are only 3

  • String()
  • Number()
  • Boolean()
    • falsy values (there are only 6)
      1. false
      2. null
      3. undefined
      4. "" or ''(the empty string)
      5. 0
      6. NaN
    • truthy values (everything else)

Implicit coercions

  • mathematical operations: +, -, *, /, %, **
  • boolean operations: ! , && , ||

Equality operators

  • == (do not use) does coercion
  • === does not allow for coercion

Short circuiting evaluations of expressions with boolean operators

  • &&
  • ||
